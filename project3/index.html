Controls

  • A => Move block left
  • D => Move block right
  • Q => Rotate block counterclockwise
  • E => Rotate block clockwise
  • M1 => Fire bullet
  • Space => Pause game

Use the mouse to move around your blaster to knock tiles off the falling blocks. Forming a line with the blocks will earn 100 points. If the majority of a line is unmodified blocks, only 1/3 points will be given.
